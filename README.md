# Fireflies Meeting Data Scraper

Scrapes meeting data from the Fireflies.ai GraphQL API to build a mapping of client companies and contacts.

## Setup

1. Create and activate the virtual environment:
   ```
   python -m venv venv
   venv\Scripts\activate      # Windows
   source venv/bin/activate   # macOS/Linux
   ```
2. Install dependencies:
   ```
   pip install -r requirements.txt
   ```
3. Copy your Fireflies API key into `.env`:
   ```
   FIREFLIES_API_KEY=your_actual_key_here
   ```
