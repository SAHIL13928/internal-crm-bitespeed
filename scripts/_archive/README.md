# scripts/_archive/

Legacy one-shot scripts from earlier iterations of this project. Kept
for git-history reference only — **do not run against production**.

These scripts pre-date the FastAPI CRM. They were used during the
"number mapping" exploration phase to build the seed CSVs that now
live in `data/inputs/`. The CSVs are the canonical input now; the
fetchers/builders here are no longer needed.

If you find yourself wanting to run one of these, check whether the
problem is better solved by a fresh script in `scripts/` proper.
